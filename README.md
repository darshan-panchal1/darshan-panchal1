## Featured Projects

### [Anatomy-of-Liability Auditor](https://github.com/darshan-panchal1/ai-anatomy-liability-auditor)
`LangGraph · FastAPI · Wikidata SPARQL · CourtListener`

4-node agentic pipeline that extracts anatomical ontology IDs via Wikidata SPARQL, retrieves US case-law precedents via CourtListener, and synthesizes structured malpractice liability reports in seconds. · [Live Demo](https://ai-anatomy-liability-auditor.netlify.app/)

### [Orbital Jurist](https://github.com/darshan-panchal1/orbital-jurist)
`LangGraph · SGP4 · Groq · RunPod Serverless · CelesTrak`

Multi-agent system fetching live TLE satellite data from CelesTrak, propagating orbital paths with SGP4, and applying maritime law analogies to adjudicate collision liability under the 1972 Outer Space Liability Convention. · [Live Demo](https://orbital-jurist.netlify.app/)

### [Slang-Vigilance](https://github.com/darshan-panchal1/slang-aware-pharmacovigilance)
`LangGraph · FastMCP · OpenFDA · RunPod`

Agentic pharmacovigilance system mapping social-media drug slang to MedDRA Preferred Terms and validating adverse drug reaction signals against FDA FAERS in real time. Deployed on RunPod Serverless. · [Live Demo](https://slang-aware-pharmacovigilance.netlify.app/)

### [Cetacean-Aware Logistics Router](https://github.com/darshan-panchal1/cetacean-router)
`LangGraph · FastAPI · Docker · RunPod · OBIS · Shapely`

3-agent LangGraph pipeline (Navigator, Marine Biologist, Risk Manager) querying live OBIS cetacean sighting data and applying a 0–100 composite scoring model to select the safest, most efficient shipping route. · [Live Demo](https://cetacean-router.netlify.app/)
